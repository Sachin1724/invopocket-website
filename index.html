Installation Guide

Install INVO Pocket
in Four Steps

Installing an APK is simple. Follow these steps to get INVO Pocket running on your Android device.

1
Download the APK
Tap the Download APK button on this page. The APK file will be saved to your device's Downloads folder.
2
Open the APK File
Go to your Downloads folder or tap the notification. Select the INVO Pocket APK file to begin installation.
3
Select Scan or Allow
Android may prompt "Scan with Play Protect" — tap Scan. If asked to allow unknown sources, tap Settings then enable Allow from this source.
4
Tap Install
Tap the Install button on the installer screen. Once complete, tap Open to launch INVO Pocket and begin your one-time setup.

Why does Android show a warning? Android displays a security prompt for any app installed outside the Play Store. INVO Pocket is completely safe. Tapping Scan or Allow simply lets the installation proceed. No account or permissions beyond storage are required.
